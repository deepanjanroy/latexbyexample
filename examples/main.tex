\begin{definition}[Unitary matrix]
A square matrix $Q \in \mathbb{C}^{m \times m} $ is unitary (in the real case, we say orthogonal) if $Q^* = Q ^{-1}$, i.e, if $Q Q^* = I$.

\end{definition}
